# Unet_SR
Dataset: DIV2K

This work mainly focus on super-resolution task using Unet.

The original paper of Unet is from https://link.springer.com/chapter/10.1007/978-3-319-24574-4_28

Apply a spatially adaptive regularization to enhance edges.

Hybrid_unet follows the logic from https://www.sciencedirect.com/science/article/pii/S0010482524003494
